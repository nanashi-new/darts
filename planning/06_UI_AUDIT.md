# 06 - UI Audit

Этот файл - рабочий чеклист экранов для fullscreen workspace UI и UI polish.

## Правило Аудита

Для каждого экрана проверяем:

- Видимый текст на русском.
- Нет старых терминов и placeholder-текста.
- Таблицы и списки получают основное место.
- Длинные формы имеют scroll/split/modal.
- Кнопки короткие, подробности в tooltip.
- Интерфейс держится на 1920x1080, 1366x768 и уменьшенном окне.

## Результаты Аудита

Пока аудит не завершен. Результаты заполнять короткими строками:

- `ok` - экран готов.
- `needs-polish` - есть точечная UI-задача.
- `blocked` - нужна среда, данные или ручная проверка.
