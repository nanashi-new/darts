# 07 - Release Notes Draft

Черновик пользовательских релизных заметок. Финальный текст переносится в release docs перед публикацией.

## v1.1.0 - Дартс Лига

### Главное

- Приложение переименовано в «Дартс Лига».
- Интерфейс переведен на русский.
- Приложение открывается в большом рабочем окне по умолчанию.
- Основные экраны готовятся под удобную работу с таблицами, турнирами и игроками.
- Старый классификационный слой удален из продукта и расчетов.

### Установка

- Основной способ установки: Windows installer `DartsLiga-Setup.exe`.
- ZIP/exe-сборка остается запасным вариантом.
- Перед релизом должен пройти clean-profile smoke.

### Для Пользователя

- Больше полезного места для рейтинга, турниров и списков игроков.
- Русские названия вкладок, кнопок, сообщений и диалогов.
- Более понятные релизные артефакты: `DartsLiga.exe`, `DartsLiga-release.zip`, `DartsLiga-Setup.exe`.

### Технические Замечания

- Старые пользовательские данные не должны мигрировать ради удаления классификации.
- Внутренние технические коды остаются техническими значениями.
- Если installer недоступен на машине сборки, ZIP/exe остается fallback.

## v1.2.0 - Удобство Ежедневной Работы

### Главное

- Безопасность данных: резервное копирование, экспорт и импорт профиля, health-check (целостность + размер + точки восстановления)
- Сезонные переходы лиг: автоматический расчёт нижних 4 Премьер-лиги / верхних 4 Первой лиги по rolling rating
- Карточка игрока v2: 6 вкладок (Общее, Рейтинг, Турниры, Заметки, Тренировки, История)
- Полировка турнирного workflow: визуальные статусы (иконки/цвета), stepper, фильтры по статусу и лиге
- Расширения рабочего пространства: теги, вложения, кастомные поля
- Тема оформления: светлая и тёмная с QSS-стилями, кастомизация (акцент, шрифт, логотип, иконка)
- Статус-бар, приветственный экран, сортировка таблиц

## v1.3.0 - Тренерские Сценарии

### Главное

- Тренерский workflow: задачи с приоритетами, статусами и сроками
- Тренировочные планы: упражнения, продолжительность, прогресс
- Вкладка «Тренер» в главном окне: подвкладки Задачи/Планы/Сводка с аналитикой (open/overdue/urgent/done)
- Интеграция тренерского плана в карточку игрока (7-я вкладка «План тренера»)
- Фильтры и быстрое завершение задач
