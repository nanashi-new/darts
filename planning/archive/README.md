# Archive

Эта папка предназначена для старых планов, которые нужно сохранить как историю, но нельзя использовать как активную очередь задач.

Активный порядок разработки начинается с `planning/00_PRIORITY.md`.

## Папки

- `root-docs/` - старые корневые документы `01_*`-`09_*` и дизайн-обзор.
- `docs-to-work/` - прежний набор master-plan/user/admin/release документов.
- `release-artifacts/` - исторические release run, smoke и manual отчеты.

Если информация из архива снова становится актуальной, ее нужно перенести в активный task-файл или один из верхнеуровневых файлов `planning/`.
