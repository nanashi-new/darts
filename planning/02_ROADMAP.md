# 02 - Roadmap

## v1.1 - Русский Release Polish ✅ Выполнено

Цель: привести текущий desktop baseline к понятному русскому релизу для Windows.

- Полная русификация видимого UI.
- Переименование продукта в «Дартс Лига».
- Удаление старой классификационной логики из бренда, UI, документации и расчетов.
- Запуск приложения в максимизированном рабочем режиме по умолчанию.
- Оптимизация основных экранов под широкий desktop: больше полезной области, меньше тесных блоков, аккуратные таблицы.
- UI polish для плотных данных: модалки деталей, скроллы, короткие кнопки и tooltip.
- Сверка с подтвержденными требованиями заказчика перед release readiness: рейтинг, импорт, турниры/лиги, отчеты и карточка игрока.
- Windows installer как основной удобный артефакт.
- ZIP/exe оставить fallback-артефактом.
- Clean-profile smoke и release checklist должны быть зелеными.

## v1.2 - Удобство Ежедневной Работы ✅ Выполнено

Цель: быстрее вести игроков, турниры и рабочий контекст.

- Главная как рабочий центр: статусы, быстрые действия, понятное начало работы.
- Безопасность данных: backup/restore, понятные предупреждения, health-check профиля.
- Карточка игрока v2: история, заметки, турниры, прогресс.
- Полировка tournament workflow: шаги турнира, состояния, публикация.
- Теги.
- Вложения.
- Кастомные поля.
- Расширенное сохранение рабочего пространства.
- Контекстные панели для ежедневной работы.

## v1.3 - Тренерские Сценарии ✅ Выполнено

Цель: развить приложение как рабочий инструмент тренера и организатора.

- Follow-up задачи.
- Тренировочные планы.
- Расширенная карточка игрока.
- Контекст тренера.
- Аналитика прогресса игрока.
- Простая очередь следующих действий по игрокам.

## v1.4 - Следующие Шаги

Идеи попадают сюда только после фиксации в `planning/00_PRIORITY.md` или отдельном task-файле.

- Импорт из дополнительных форматов
- Настраиваемые отчёты
- Мультипрофильность
- Автообновление
