# Task: Coach Workflow v1

## Цель

Развить приложение как инструмент тренера: задачи, follow-up, планы и контекст игрока.

## Что Меняем

- Добавить структуру тренерских follow-up задач.
- Связать тренировочные планы с игроком.
- Подготовить контекст тренера: что было, что сделать дальше, какие заметки важны.

## Что Не Трогаем

- Не смешивать тренерские задачи с турнирами.
- Не добавлять сложную аналитику до базового workflow.
- Не менять rating core.

## Тесты И Ручные Проверки

- Тесты хранения и отображения follow-up задач.
- UI smoke тренерского контекста.
- Проверка пустого состояния и игрока с большим количеством заметок.

## Критерии Готовности

- Тренер видит следующий шаг по игроку.
- Тренировочный контекст не теряется между сессиями.
- Workflow остается простым и не требует отдельного обучения.
