# Task: Автоматические сезонные переходы лиг v2

Статус: planned  
Приоритет: P1  
Источник: утвержденные требования заказчика, вынесено из v1.1 P0.

## Цель

Добавить автоматический расчет сезонного обмена между Премьер-лигой и Первой лигой по правилу “нижние 4 Премьер / верхние 4 Первой”.

## Что меняем

- Рассчитать кандидатов на переход по итогам выбранного периода/сезона.
- Показать preview до применения переходов.
- Применять переходы только после явного подтверждения пользователя.
- Записывать transfer history и audit для каждого игрока.

## Что не трогаем

- Не меняем v1.1 режим: при публикации турнира уже есть preview/history по фактической лиге турнира.
- Не выполняем скрытые сезонные переходы без подтверждения.
- Не смешиваем взрослые league ratings с детскими category ratings.

## Тесты и проверки

- Unit/integration tests для выбора нижних 4 и верхних 4.
- Edge cases: меньше 4 игроков в лиге, равенство очков, пустой сезон.
- UI smoke для preview и подтверждения.

## Критерии готовности

- Пользователь видит список кандидатов до применения.
- Применение создает history/audit.
- Текущий v1.1 publish-flow не ломается.
