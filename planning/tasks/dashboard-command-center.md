# Task: Dashboard Command Center

## Цель

Сделать вкладку `Главная` рабочим центром приложения, а не декоративной стартовой страницей.

## Что Меняем

- Показать актуальный статус профиля, базы, последнего импорта и релизных проверок.
- Добавить быстрые действия для частых сценариев: открыть рейтинг, создать/открыть турнир, перейти к игрокам, импортировать данные.
- Сделать блоки компактными и рассчитанными на maximized workspace.

## Что Не Трогаем

- Бизнес-логику рейтинга и турниров.
- Формат БД.
- Новый дизайн бренда.

## Тесты И Ручные Проверки

- UI smoke: вкладка создается, текст на русском, нет старого бренда.
- Ручная проверка на 1920x1080 и 1366x768.
- Проверить, что быстрые действия ведут на существующие вкладки/сценарии.

## Критерии Готовности

- Главная помогает начать работу без чтения инструкции.
- Быстрые действия не дублируют хаотично весь интерфейс.
- Экран не превращается в лендинг.
