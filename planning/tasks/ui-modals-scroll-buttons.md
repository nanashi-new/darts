# Task: UI Polish - Модалки, Скроллы, Кнопки

Статус: done
Приоритет: P0

## Цель

Сделать интерфейс удобным для плотных данных: раскрытие строк в модалках, скроллинг в перегруженных местах, кнопки без обрезанных надписей.

Эта задача выполняется после `fullscreen-workspace-ui.md`: сначала фиксируется широкий рабочий режим приложения, затем доводятся отдельные перегруженные места.

## Что Меняем

- Добавить отдельные модалки подробного просмотра для строк турниров, игроков, рейтинга, импорта и диагностики.
- Добавить двойной клик или кнопку `Открыть` для таблиц, где строка содержит много данных.
- Добавить scroll containers в формы/панели, где контент может не помещаться.
- Сократить длинные подписи кнопок и добавить tooltip.
- Задать устойчивые `minimumWidth`, `sizePolicy`, переносы и resize behavior для русских текстов.
- На широком экране использовать доступное место для таблиц, сплиттеров и подробных панелей, а не оставлять большие пустые зоны.

## Что Не Трогаем

- Бизнес-логику расчетов.
- Формат данных и БД.
- Релизные scripts.

## Тесты И Проверки

- UI smoke на создание основных экранов.
- Ручная проверка максимизированного окна, маленького окна и длинных русских строк.
- Проверка, что модалки открываются из таблиц и не теряют данные.
- Проверка, что текст не вылезает за кнопки.

## Готово, Если

- Длинные данные можно открыть в отдельной модалке.
- Экраны не ломаются на маленьком размере окна.
- Кнопки выглядят аккуратно с русским текстом.
