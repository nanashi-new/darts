# Task: Future Workspace Features

## Цель

Спланировать расширение ежедневной работы после v1.1.

## Что Меняем

- Теги для игроков, турниров и заметок.
- Вложения.
- Кастомные поля.
- Расширенное сохранение рабочего пространства.
- Улучшенные карточки и тренерский контекст.

## Что Не Трогаем

- До v1.1 не внедрять эти функции в код.
- Не менять схему БД без отдельного migration/task-файла.

## Тесты И Проверки

- Для будущей реализации потребуется migration coverage.
- UI smoke для новых полей и вложений.
- Экспорт/импорт новых данных.

## Готово, Если

- После v1.1 каждая функция разбита на отдельные P1/P2 task-файлы с миграциями и acceptance criteria.
