# Task: Небольшая автоматизация v1

Статус: done  
Приоритет: P0  
Источник: следующий план разработки и утвержденные требования заказчика.

## Цель

Добавить аккуратные автоматические помощники, которые ускоряют работу администратора, но не меняют критичные данные молча.

## Что меняем

- Автопересчет после публикации/исправления турнира.
- Автоснимок рейтинга при публикации.
- Автосоздание restore point перед опасными действиями.
- Preview переходов между Премьер-лигой и Первой лигой до публикации.
- Подсказка категории по дате рождения и дате турнира.
- Понятные уведомления после импорта: принято, ошибки, следующий шаг.

## Что не трогаем

- Не публикуем импорт автоматически без review-flow.
- Не меняем категорию, лигу или результаты скрыто от пользователя.
- Не делаем физическое удаление турниров обычным действием без отдельного решения.

## Тесты и проверки

- Targeted tests для подсказки категории.
- Targeted tests для snapshots/recalc/restore/league preview там, где меняется код.
- UI smoke для import review и подсказок.

## Критерии готовности

- Категория показывается как подсказка, а не скрытая запись в данные.
- Публикация и correction-flow создают понятные rating/snapshot/audit следы.
- Опасные действия имеют restore/safe path.
- Preview переходов лиг понятен до публикации.
