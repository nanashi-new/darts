# Task: Стабилизировать Русификацию UI

## Цель

Закрыть текущую русификацию до состояния, где весь видимый UI на русском и это закреплено тестами.

## Что Меняем

- Завершить `app/ui/labels.py` для статусов, scopes, заметок, visibility, priority, audit events, уровней логов, пола.
- Применить label helpers во всех таблицах и карточках, где сейчас могут светиться технические коды.
- Убрать оставшиеся видимые английские строки вроде `Excel Files`.
- Сохранить aliases для старого сохраненного UI-state.
- Держать `PDF`, `XLSX`, `PNG`, `JSON`, env vars и пути как технические значения.

## Что Не Трогаем

- Схему БД.
- Внутренние enum/code значения.
- Исторические docs artifacts.

## Тесты И Проверки

- `tests/test_ui_labels.py`
- `tests/test_ui_tabs_content_smoke.py`
- `tests/test_ui_russian_text.py`
- `tests/test_ui_state.py`
- `rg` по запрещенным видимым английским фразам.

## Готово, Если

- Верхние вкладки русские.
- Видимые таблицы/диалоги не показывают английские рабочие подписи.
- Тесты русификации проходят или блокер запуска явно зафиксирован.
