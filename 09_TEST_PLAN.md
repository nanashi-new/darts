## 2. Тесты классификации (разряды)

- результат ниже норматива → 0
- минимально выполненный разряд → корректные очки
- при нескольких дисциплинах → сумма
- до 10 лет → учитывается только «Набор очков»

---

## 3. Тесты rolling-рейтинга

### N = 3
- у игрока 5 турниров → берутся 3 последних по дате

### N = 12
- у игрока 4 турнира → берутся все 4

---

## 4. Тесты категорий

- мальчики и девочки считаются отдельно
- разные возрастные категории не смешиваются
- при переходе по возрасту:
  - старая категория не влияет
  - новая категория считается с нуля

---

## 5. Тесты импорта Excel

### Успешные сценарии
- один файл, одна таблица
- один файл, несколько таблиц подряд
- импорт папки (10+ файлов)

### Ошибки
- отсутствует колонка ФИО
- некорректные числовые значения
- пустые строки

Ожидание:
- понятное сообщение
- приложение не падает

---

## 6. Тесты экспорта

- PDF открывается и не обрезается
- Excel открывается в MS Excel
- PNG соответствует экрану

---

## 7. Критерий приёмки

Приложение принимается, если:
- все тесты пройдены;
- расчёты совпадают с эталонной ручной проверкой;
- импорт/экспорт стабилен;
- база данных переносится без потерь.
