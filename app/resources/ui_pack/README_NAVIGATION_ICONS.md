# Дартс Рейтинг — навигационные иконки

Утверждённый стиль:
- строгий Windows desktop/software UI;
- графитовый контур + синий основной акцент + красный точечный акцент;
- без киберспорта, щитов, кубков, фотореализма;
- SVG как основной формат, PNG только для предпросмотра/быстрой вставки.

## Палитра
- Graphite: #1F2328
- Blue: #1D3F8C
- Red: #D52B1E
- Background: #F7F9FC
- Gray: #8A94A6
