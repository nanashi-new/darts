# Руководство пользователя Дартс Лига

## Начало работы

1. При первом запуске приложение создает профиль данных по умолчанию.
2. Все данные хранятся локально в папке профиля.
3. Перейдите во вкладку «Настройки», чтобы задать параметры отображения.
4. Используйте вкладку «Импорт/Экспорт» для загрузки турнирных данных.
5. Для создания дополнительных профилей откройте «Управление профилями» в Настройках.

## Управление игроками

1. Откройте вкладку «Игроки» для просмотра списка всех участников.
2. Для добавления игрока нажмите кнопку «Добавить игрока» и заполните форму.
3. Для редактирования данных дважды щелкните по строке игрока.
4. Используйте поле поиска для быстрого нахождения игрока по имени.
5. Карточка игрока отображает историю участия и текущий рейтинг.

## Турниры

1. Как провести турнир через XLSX: откройте «Импорт/Экспорт», нажмите «Импорт файла» и выберите XLSX.
2. После импорта данные проходят проверку на корректность.
3. Что делать перед публикацией: просмотрите предупреждения и исправьте ошибки.
4. Нажмите «Опубликовать», чтобы результаты вошли в рейтинг.
5. Для ручного создания турнира используйте вкладку «Турниры».
6. Коррекция результатов доступна до момента публикации.

## Рейтинг

1. Рейтинг рассчитывается автоматически на основе опубликованных турниров.
2. Учитываются N лучших результатов каждого игрока (параметр настраивается).
3. Вкладка «Рейтинг» позволяет фильтровать по категориям и лигам.
4. Баллы начисляются в зависимости от занятого места и количества участников.
5. При обновлении рейтинга создается снимок для отслеживания изменений.

## Тренерская работа

1. Вкладка «Тренер» предназначена для управления тренировочным процессом.
2. Создавайте задания и планы для отдельных игроков.
3. Отслеживайте выполнение задач и прогресс тренировок.
4. Ведите тренировочный журнал с заметками.
5. Используйте фильтры для просмотра заметок по типу и видимости.

## Резервные копии и безопасность

1. Перед критическими операциями создавайте точку восстановления.
2. Для создания резервной копии используйте «Экспорт» во вкладке «Импорт/Экспорт».
3. Восстановление из копии доступно через кнопку «Импорт».
4. Как восстановиться после ошибки: откройте «Диагностика», выберите точку восстановления.
5. Регулярно делайте резервные копии, особенно перед обновлением приложения.

## Настройка внешнего вида

1. Откройте вкладку «Настройки» для изменения оформления.
2. Выберите тему: светлая или темная.
3. Настройте акцентный цвет для выделения элементов интерфейса.
4. Измените размер шрифта: мелкий, средний или крупный.
5. Загрузите собственный логотип организации для отображения в приложении.

## Горячие клавиши и подсказки

1. Ctrl+1..9 - быстрое переключение между вкладками.
2. Ctrl+I - открыть импорт данных.
3. F1 - открыть справку.
4. Наведите курсор на любую кнопку для отображения подсказки.
5. Используйте поле поиска в Справке для быстрого поиска нужной информации.
