# 12 — Сборка Windows (.exe)

## Что делает сборка
Сборка формирует Windows-версию программы `DartsRatingEBCK.exe` без консольного окна.

### Важные условия
- Программа запускается **без консоли**.
- Рабочие данные хранятся в профиле пользователя Windows (`%APPDATA%\\Darts`).
- Файл ресурсов `norms.xlsx.b64` включён в сборку.

## Что нужно установить на Windows
1. Python 3.12+
2. Зависимости проекта:
   ```bat
   python -m pip install -r requirements.txt
   ```
3. PyInstaller:
   ```bat
   python -m pip install pyinstaller
   ```

## Быстрый запуск в режиме разработки
```bat
scripts\run_dev.bat
```

## Сборка .exe
```bat
scripts\build.bat
```

После успешной сборки готовый запускной файл будет в папке:
- `dist\DartsRatingEBCK\DartsRatingEBCK.exe`

## Проверка после сборки
1. Запустите `DartsRatingEBCK.exe`.
2. Убедитесь, что не открывается консольное окно.
3. Проверьте, что в `%APPDATA%\\Darts` появились:
   - база данных `app.db`;
   - `settings.json`;
   - рабочие файлы журнала/экспорта/профилей по сценарию использования.
4. Откройте программу и проверьте импорт/пересчёт/экспорт на тестовом файле.
