# 12 — Сборка Windows (`.exe`)

## Что делает сборка

Сборка формирует Windows-версию программы `DartsLiga.exe` без консольного окна.

## Важные условия

- Программа запускается **без консоли**.
- Рабочие данные хранятся в профиле пользователя Windows: `%APPDATA%\<папка приложения>\`.
- Фактическое имя папки приложения смотрите в логах или в настройках приложения.
- Дополнительные пользовательские данные хранятся вне исполняемого файла.

## Что нужно установить на Windows

- Python **3.11.x** (целевая и рекомендуемая версия).
- Зависимости проекта.
- PyInstaller.

## Команды установки

### Установка зависимостей проекта

```bat
python -m pip install -r requirements-pinned.txt
```

### Установка PyInstaller

```bat
python -m pip install pyinstaller
```

## Запуск в режиме разработки

```bat
scripts\run_dev.bat
```

## Сборка `.exe`

```bat
scripts\build.bat
```

> Для release/offline-сборки используется только `requirements-pinned.txt`.
> `requirements.txt` остаётся для dev-сценариев.

После успешной сборки исполняемый файл находится по пути:

```text
dist\DartsLiga\DartsLiga.exe
```

Спецификация сборки PyInstaller находится в корне проекта:

```text
pyinstaller.spec
```

## Проверка после сборки

1. Запустите `DartsLiga.exe`.
2. Убедитесь, что **консольное окно не открывается**.
3. Проверьте, что в папке `%APPDATA%\<папка приложения>\` появились:
   - база данных `app.db`;
   - файл настроек `settings.json`;
   - файлы журнала;
   - папки экспорта и профилей (по сценарию использования).
4. Откройте программу и проверьте:
   - импорт Excel-файла;
   - пересчёт турнира;
   - экспорт отчёта.
