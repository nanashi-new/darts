# 12 — Сборка Windows (`.exe`)

## Что делает сборка

Сборка формирует Windows-версию программы `DartsRatingEBCK.exe` без консольного окна.

## Важные условия

- Программа запускается **без консоли**.
- Рабочие данные хранятся в профиле пользователя Windows:  
  `%APPDATA%\<папка приложения>\`
- Фактическое имя папки приложения смотрите в логах или в настройках приложения.
- Файл ресурсов `norms.xlsx.b64` включается в сборку.

## Что нужно установить на Windows

- Python **3.12+**
- Зависимости проекта
- PyInstaller

## Команды установки

### Установка зависимостей проекта

```bat
python -m pip install -r requirements.txt
```

### Установка PyInstaller

```bat
python -m pip install pyinstaller
```

## Быстрый запуск в режиме разработки

```bat
scripts\run_dev.bat
```

## Сборка `.exe`

```bat
scripts\build.bat
```

После успешной сборки исполняемый файл находится по пути:

- `dist\DartsRatingEBCK\DartsRatingEBCK.exe`

Спецификация сборки PyInstaller находится в корне проекта: `pyinstaller.spec`.

## Проверка после сборки

- Запустите `DartsRatingEBCK.exe`.
- Убедитесь, что консольное окно не открывается.
- Проверьте, что в `%APPDATA%\<папка приложения>\` появились:
  - база данных `app.db`;
  - `settings.json`;
  - рабочие файлы журнала, экспорта и профилей (по сценарию использования).
- Откройте программу и проверьте импорт, пересчёт и экспорт на тестовом файле.
