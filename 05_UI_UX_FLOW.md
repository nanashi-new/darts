## 1. Общие принципы интерфейса

- минимальное количество кликов;
- все основные действия — в 1–2 шага;
- крупные таблицы, читаемые шрифты;
- одинаковое поведение кнопок во всех разделах;
- подсказки и FAQ доступны **из любого экрана**.

## 3. Экран «Рейтинг»

### Назначение
Просмотр текущего рейтинга EBCK.

### Элементы
- выпадающий список категории;
- фильтр по полу;
- фильтр по лиге (для взрослых);
- поле выбора `N` (3…12);
- поле быстрого поиска по ФИО.

### Таблица
Колонки:
- место;
- ФИО;
- очки рейтинга;
- количество учтённых турниров.

### Кнопки
- «Экспорт в Excel»
- «Экспорт в PDF»
- «Печать»
- «Сохранить как изображение»

---

## 4. Экран «Турниры»

### 4.1 Список турниров
- таблица: дата / название / категория / лига;
- поиск по названию;
- кнопки: Добавить / Открыть / Удалить.

---

### 4.2 Карточка турнира

#### Верхняя часть
- название турнира;
- дата;
- категория;
- лига.

#### Кнопки
- «Импорт XLSX»
- «Добавить строку вручную»
- «Пересчитать»
- «Экспорт протокола»

#### Таблица результатов
- ФИО
- дисциплины
- разряды
- очки классификации
- очки за место
- итог турнира

Любое изменение строки → кнопка «Пересчитать».

---

## 5. Экран «Игроки»

### 5.1 Список игроков
- поиск по ФИО;
- фильтр по полу;
- фильтр по тренеру;
- кнопки: Добавить / Открыть.

---

### 5.2 Карточка игрока

Показывает:
- персональные данные;
- список турниров;
- очки по каждому турниру;
- итоговый рейтинг.

(v1.1)
- график изменения очков;
- сравнение с другим игроком.

---

## 6. Экран «Импорт / Экспорт»

### Импорт
- кнопка «Импорт файла»
- кнопка «Импорт папки»
- список последних импортов
- журнал ошибок

### Экспорт
- экспорт базы данных (бэкап);
- импорт базы данных;
- пакетный экспорт отчётов.

---

## 7. Экран «Отчёты»

- рейтинг по всем категориям;
- топ-N игроков;
- отчёт по тренеру;
- отчёт по турниру.

## 8. FAQ / Помощь

### Назначение
Объяснить пользователю:
- куда нажимать;
- что означают поля;
- почему получился такой результат.

### Реализация
- отображение `FAQ.txt`;
- поиск по словам;
- кликабельные якоря;
- кнопка «Открыть соответствующий экран».

---

## 9. Настройки

- значение N (по умолчанию);
- путь бэкапов;
- формат экспорта;
- включение/отключение подтверждений;
- версия нормативов ЕВСК.
